1st place — Aggies Create Innovation Expo

AggiesBCI is a brain–computer interface that lets a user steer an electric wheelchair using trained mental commands — no joystick, no physical input. We built it as a universal module that clamps onto an existing e-wheelchair, reading EEG signals and translating intent into motion. The project took 1st place at the Aggies Create Innovation Expo.
